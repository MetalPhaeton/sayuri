43 How To Build with CMake
44 -----------------------
45 
46 To use CMake, you can build Sayuri easily.
47 
48 It's default compiler is &quot;clang++&quot;.
49 
50 (If you want to change a compiler, please change 26th line and 27th line
51 in &quot;CMakeLists.txt&quot;.)
52 
53 ###### To build in normal mode ######
54 
55 1. Change directory to where &quot;CMakeLists.txt&quot; is placed in.
56 2. Run the following commands.
57  1. $ mkdir build
58  2. $ cd build
59  3. $ cmake ..
60  4. $ make
61 3. The following files or directories have created in &quot;build&quot; directory.
62  * sayuri : A chess engine for UCI.
63  * libsayurilib.a : A static library.
64  * include : A directory for header files for &quot;libsayurilib.a&quot;.
65 
66 ###### To build in debug mode ######
67 
68 To build in debug mode, you can build Sayuri with &quot;No optimized&quot; and
69 &quot;Embedded the infomation for debugger&quot;.
70 
71 In the following 2-3, insert &quot;-DCMAKE\_BUILD\_TYPE=Debug&quot; only.
72 
73 1. Change directory to where &quot;CMakeLists.txt&quot; is placed in.
74 2. Run the following commands.
75  1. $ mkdir build
76  2. $ cd build
77  3. $ cmake -DCMAKE\_BUILD\_TYPE=Debug ..
78  4. $ make
79 3. The following files or directories have created in &quot;build&quot; directory.
80  * sayuri : A chess engine for UCI.
81  * libsayurilib.a : A static library.
82  * include : A directory for header files for &quot;libsayurilib.a&quot;.
83 
84 ###### About each file ######
85 
86 &quot;sayuri&quot; is a chess engine for UCI.
87 
88 When you use it, you don't need &quot;libsayurilib.a&quot; and
89 &quot;include&quot;.
90 
91 &quot;libsayurilib.a&quot; and &quot;include&quot; are files for using Sayuri as
92 a static library.
93 When you use them, you don't need &quot;sayuri&quot;.
94 Please read
95 &quot;[How to use Sayuri as static library](@ref use_as_static_library-en.md)&quot; for details.

115 UCI Options
116 -----------
117 
118 You can change Sayuri's settings with the following commands.
119 
120 * To change size of the hash table. (Default: 32 MB, Max: 8192 MB, Min: 8 MB)
121  * setoption name Hash value <Size(MB)>
122 
123 * To initialize the hash table.
124  * setoption name Clear Hash
125 
126 * To enable Ponder. (Default: true)
127  * setoption name Ponder value <true or false>
128 
129 * To change the number of threads. (Default: 1, Max: 64, Min: 1)
130  * setoption name Threads value <Number of threads>
131 
132 * To enable analyse mode. (Default: false)
133  * setoption name UCI\_AnalyseMode value <true or false>
